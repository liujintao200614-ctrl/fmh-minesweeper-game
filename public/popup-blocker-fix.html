🚫➡️✅ MetaMask 弹窗修复工具

🚨 弹窗被阻止的常见原因

主要问题：浏览器阻止了 MetaMask 弹窗显示

⚠️ 重要：这通常发生在：
  • 浏览器弹窗阻止设置过于严格
  • MetaMask 扩展被禁用或有问题
  • 网站没有用户交互就尝试连接
  • 浏览器安全策略阻止

🔧 快速修复方案

Chrome 浏览器修复

1. 检查弹窗阻止设置
地址栏右侧看是否有 🚫 图标，点击并选择"始终允许弹出式窗口"
2. 手动允许弹窗
Chrome设置 → 隐私设置和安全性 → 网站设置 → 弹出式窗口和重定向 → 允许
3. 检查 MetaMask 扩展
chrome://extensions/ → 找到 MetaMask → 确保已启用

Firefox 浏览器修复

1. 弹窗设置
地址栏左侧盾牌图标 → 弹出窗口已阻止 → 允许
2. 全局设置
设置 → 隐私与安全 → 权限 → 阻止弹出窗口 → 例外 → 添加本站点

Safari 浏览器修复

1. Safari 设置
Safari → 偏好设置 → 网站 → 弹出式窗口 → 允许
2. 检查扩展
Safari → 偏好设置 → 扩展 → 确保 MetaMask 已启用

Edge 浏览器修复

1. 弹窗设置
地址栏右侧 🚫 → 允许弹出窗口和重定向
2. 全局设置
设置 → Cookies和网站权限 → 弹出窗口和重定向 → 添加允许列表

🧪 立即测试修复效果

✅ 高级修复方案

方案1: 直接打开 MetaMask

如果弹窗不出现，可以直接点击浏览器工具栏的 MetaMask 图标

方案2: 使用用户手势触发

确保连接请求是由用户点击触发的，而不是页面自动执行

方案3: 检查权限

🔄 终极解决方案

如果以上都不行，尝试：
  1. 重启浏览器 - 完全关闭再重新打开
  2. 禁用其他扩展 - 临时禁用广告拦截器等
  3. 无痕模式测试 - 在隐私/无痕窗口中测试
  4. 清除缓存 - 清除浏览器缓存和 Cookie
  5. 重装 MetaMask - 卸载后重新安装扩展
